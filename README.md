# DC Motor Controller Firmware for STM32F107/DRV8701

My first STM32 project. Starting with arduino framework and moving over to baremetal later as i 
learn how the stm32 works...

## Features

- Rotary encoder, 3 push buttons, and 125x240 TFT display
- DRV8701 motor driver (12-36V, 80A)
- PID motor controller
- Overcurrent protection
- MT6701 magnetic encoder with up to 1024 PPR support
- LED constant current driver
